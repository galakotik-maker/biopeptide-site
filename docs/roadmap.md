# Roadmap

## Phase 1: Foundations
- Finalize data model and architecture.
- Choose runtime stack (Node or Python).
- Create Supabase project and schema.

## Phase 2: Ingestion
- Implement PubMed ingestion and deduplication.
- Build embeddings pipeline and retrieval.

## Phase 3: Generation
- Medical Writer drafts with citations.
- Creative Editor produces blog-ready content.

## Phase 4: Bots
- Telegram Bot Gateway with Moderator.
- Dr Drake protocol delivery and audit logs.

## Phase 5: Product
- Lovable UI integration via Supabase.
- Admin console for review and approvals.
